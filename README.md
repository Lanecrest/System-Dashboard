# System Dashboard Matrix
This is a cross-platform system monitoring dashboard using Python and Flask front-end. 
Many of the processes rely on the non-stock psutil module as well as other built in modules.

# About
The idea is to be able to spin this up on a machine and get system information/vitals 
at a glance. Development is being tested on Windows 11, macOS (Silicon) and Linux. 
Cross-compatibility is a key development goal while also maintaining a "retro" aesthetic.
